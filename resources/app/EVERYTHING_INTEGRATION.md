# Everything 集成使用说明

## 功能简介

将 Everything 搜索工具集成到日志查看器中，可以快速搜索和加载日志文件，大幅提高工作效率！

## 配置步骤

### 1. 安装 Everything

如果还没有安装 Everything，请从官网下载：
- 官网：https://www.voidtools.com/
- 下载地址：https://www.voidtools.com/downloads/

### 2. 启用 Everything HTTP 服务器

**方法一：通过 Everything 界面启用**
1. 打开 Everything
2. 菜单 → 工具 → 选项 → HTTP 服务器
3. 勾选"启用 HTTP 服务器"
4. 默认端口：8888（保持默认即可）
5. 点击"确定"

**方法二：命令行启动**
```bash
# 关闭现有 Everything 实例
# 然后使用 -http 参数启动
Everything.exe -http
```

### 3. 验证连接

启动日志工具后，如果看到搜索框提示：
- ✓ "Everything 快速搜索 (支持通配符 *) ✓" → 连接成功
- ✗ "Everything 未连接，请启动 Everything HTTP 服务器" → 请检查 Everything 是否运行

## 使用方法

### 基础搜索

1. 在文件树顶部的搜索框中输入关键词
2. 支持的搜索模式：
   - `kernel_log` → 搜索包含 kernel_log 的文件
   - `*.log` → 搜索所有 .log 文件
   - `2025*` → 搜索以 2025 开头的文件
   - `*error*` → 搜索包含 error 的文件

3. 搜索结果按**修改时间降序**排列，最新的文件在最前面

### 高级搜索

| 通配符 | 说明 | 示例 |
|--------|------|------|
| `*` | 匹配任意字符 | `*.log` |
| `?` | 匹配单个字符 | `log?.txt` |
| `|` | 或操作 | `log\|txt` |
| `!` | 非 | `*.log!\*debug*` |

### 常用场景

**场景1：快速定位最新的错误日志**
```
输入：error
结果：显示所有包含 error 的文件，按时间排序
点击：直接加载到日志查看器
```

**场景2：查找特定日期的日志**
```
输入：*20250126*
结果：显示所有包含 20250126 的文件
```

**场景3：搜索特定路径的日志**
```
输入：C:\logs\*.log
结果：只搜索 C:\logs\ 目录下的日志文件
```

## 搜索结果操作

### 点击加载文件
- 点击搜索结果中的任意文件
- 自动加载到日志查看器
- 支持所有日志格式（.log, .txt, .out, .err 等）

### 文件信息展示
- **文件名**：完整的文件名
- **路径**：文件所在目录（超长路径会截断）
- **大小**：自动格式化（B/KB/MB/GB）
- **修改时间**：本地时间格式

## 常见问题

### Q: 搜索框显示"Everything 未连接"
**A:**
1. 检查 Everything 是否正在运行
2. 检查 HTTP 服务器是否已启用（端口 8888）
3. 尝试重启 Everything 和日志工具

### Q: 搜索不到某些文件
**A:**
1. 确认 Everything 已索引这些文件（在 Everything 中搜索验证）
2. 检查文件路径是否在 Everything 的索引范围内
3. 路径中包含特殊字符可能需要转义

### Q: 搜索速度慢
**A:**
1. Everything 首次启动需要建立索引
2. 检查是否在索引大量网络驱动器
3. 在 Everything 选项中调整索引范围

### Q: 能否搜索网络驱动器
**A:**
可以，但需要在 Everything 中配置：
- 菜单 → 工具 → 选项 → 索引 → 网络
- 添加需要索引的网络驱动器

## 技术细节

### API 端点
```
GET http://localhost:8888/?s=<query>&j=1&count=<limit>
```

### 支持的日志格式
- `.log` - 标准日志文件
- `.txt` - 文本日志
- `.out` - 输出日志
- `.err` - 错误日志
- `.debug` - 调试日志
- `.trace` - 追踪日志

### 快捷键
- `Enter` - 执行搜索
- `Esc` - 清空搜索
- 点击关闭按钮(×) - 收起搜索框
