# Everything 集成快速开始指南

## 🚀 3步完成配置

### 步骤 1：安装 Everything（如果还没有）

1. 下载：https://www.voidtools.com/downloads/
2. 安装到默认位置：`C:\Program Files\Everything\`
3. 启动 Everything（保持运行）

### 步骤 2：重启日志工具

工具会自动检测 `es.exe`

### 步骤 3：开始搜索

在文件树顶部的搜索框输入关键词，例如：
- `kernel` - 搜索包含 kernel 的文件
- `*.log` - 搜索所有 .log 文件
- `20260121*` - 搜索特定日期的文件

---

## ✅ 验证配置

启动日志工具后，如果搜索框提示：
- ✓ "Everything 快速搜索 (支持通配符 *) ✓" → 成功
- ✗ "Everything CLI 未就绪，请检查 es.exe" → 需要检查配置
